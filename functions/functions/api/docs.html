TitleApp API Reference

The TitleApp API lets you read and write data to any TitleApp workspace. Push leads from your website. Sync inventory from your DMS. Power a ChatGPT GPT or Claude MCP integration. Build anything.

Base URL https://publicapi-feyfibglbq-uc.a.run.app

Authentication

Every request (except /v1/health and /v1/docs) requires an API key in the X-API-Key header. Keys are scoped to specific workspaces and permission levels.

Step 1 — Generate a key

Call the createApiKey endpoint with your Firebase Bearer token. You choose which workspaces and scopes the key gets.

curl -X POST https://createapikey-feyfibglbq-uc.a.run.app \
  -H "Authorization: Bearer YOUR_FIREBASE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Website Lead Form",
    "workspace_ids": ["ws_1708901234_a8kq2x"],
    "scopes": ["read", "write"]
  }'

// Response
{
  "ok": true,
  "api_key": {
    "id": "Kj8x2mPqR4nL",
    "key": "ta_a1b2c3d4e5f6...",
    "name": "Website Lead Form",
    "workspace_ids": ["ws_1708901234_a8kq2x"],
    "scopes": ["read", "write"],
    "rate_limit": 100
  }
}

Step 2 — Use the key

curl https://publicapi-feyfibglbq-uc.a.run.app/v1/workspaces \
  -H "X-API-Key: ta_a1b2c3d4e5f6..."

Key properties

FieldDescription
keyThe API key string. Prefixed with ta_. Store it securely — it cannot be retrieved after creation.
workspace_idsArray of workspace IDs this key can access. Use "vault" for the personal vault.
scopesread — GET endpoints only. write — GET + POST. admin — full access to all workspaces.
rate_limitMax requests per hour. Default 100 (free tier).

Rate Limiting

Every response includes rate limit headers. When you hit the limit, you get a 429 until the hourly window resets.

HeaderExampleDescription
X-RateLimit-Limit100Max requests per hour for this key
X-RateLimit-Remaining73Requests left in the current window
X-RateLimit-Reset1740355200Unix timestamp when the window resets
Free tier: 100 requests per hour. Pro tier: 10,000 per hour. Need more? Contact us.

Errors

All errors return a consistent JSON structure with an error code, human-readable message, and HTTP status.

{
  "error": {
    "code": "forbidden",
    "message": "Access denied to this workspace",
    "status": 403
  }
}
StatusCodeWhat happenedHow to fix it
400bad_requestMissing or invalid field in the request bodyCheck required fields in the endpoint docs below
401unauthorizedNo API key provided, or the key is invalid/revokedAdd X-API-Key header. Generate a key at the createApiKey endpoint.
403forbiddenKey exists but lacks access to this workspace or scopeCheck workspace_ids and scopes on your key
404not_foundWorkspace, resource, or route does not existVerify the workspace ID and endpoint path
429rate_limitedExceeded the hourly request limitWait for the reset time in X-RateLimit-Reset header
500internal_errorSomething broke on our endRetry after a few seconds. If persistent, contact support.

Quick Start / SDK

Official SDKs are coming. In the meantime, here is a lightweight wrapper you can drop into any JavaScript project.

const titleapp = {
  baseUrl: 'https://publicapi-feyfibglbq-uc.a.run.app/v1',
  apiKey: 'ta_your_key_here',

  async get(path) {
    const res = await fetch(this.baseUrl + path, {
      headers: { 'X-API-Key': this.apiKey }
    });
    return res.json();
  },

  async post(path, data) {
    const res = await fetch(this.baseUrl + path, {
      method: 'POST',
      headers: { 'X-API-Key': this.apiKey, 'Content-Type': 'application/json' },
      body: JSON.stringify(data)
    });
    return res.json();
  },

  async del(path) {
    const res = await fetch(this.baseUrl + path, {
      method: 'DELETE',
      headers: { 'X-API-Key': this.apiKey }
    });
    return res.json();
  }
};

// Get dashboard KPIs
const dash = await titleapp.get('/workspaces/ws_abc123/dashboard');
console.log(dash.data.kpis.total_inventory); // 47

// Push a buyer lead from Zillow
await titleapp.post('/workspaces/ws_abc123/buyers', {
  name: 'Jennifer Kim',
  email: 'jennifer.kim@gmail.com',
  source: 'zillow',
  message: 'Interested in 1893 San Jose Blvd, pre-approved up to $475k'
});

// List available inventory
const inv = await titleapp.get('/workspaces/ws_abc123/inventory?status=available');

Workspaces

Workspaces are the top-level container. Every API key is scoped to one or more workspaces. Each workspace has a vertical (auto, real_estate, analyst, consumer) and its own data.

GET /v1/workspaces List accessible workspaces ▶
Use this to discover which workspaces your API key has access to. Call this first when setting up a new integration to get the workspace IDs you need for all other endpoints.

Response

    { "id": "ws_1709442871_m3jc7p" },
    { "id": "vault" }

Integration

Universal inbound data push and webhook subscriptions for real-time event notifications.

POST/v1/workspaces/:workspace_id/inboundUniversal data push▶
write scope
Don't want to map to a specific endpoint? POST any JSON here and the AI figures out what to do with it. Flight logs from ForeFlight, form submissions from Typeform, IoT sensor data — anything.

Request body

FieldTypeDescription
sourcestringoptionalWhere this data came from (default: unknown)
...anyAny additional JSON fields you want

Response (201)

{
  "data": {
    "id": "inb_7kM2nQ",
    "status": "pending",
    "message": "Data received and queued for AI processing"
  }
}

POST/v1/webhooksCreate webhook▶
write scope
Subscribe to real-time events. When something happens in a workspace, we POST a JSON payload to your URL.

Request body

FieldTypeDescription
urlstringrequiredHTTPS endpoint to receive events
eventsarrayrequiredEvent types to subscribe to (see below)
workspace_idstringoptionalScope to a specific workspace

Response (201)

{
  "data": {
    "id": "wh_4mQ9",
    "url": "https://your-app.com/webhooks/titleapp",
    "events": ["lead.new", "tenant.late"],
    "workspace_id": "ws_1708901234_a8kq2x",
    "status": "active"
  }
}
DELETE/v1/webhooks/:idDelete webhook▶
Remove a webhook subscription. You can only delete webhooks that belong to your API key.

Response

{ "data": { "id": "wh_3kN8", "deleted": true } }
