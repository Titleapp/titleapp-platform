# Aviation Vertical

**Suites:** Operator (Part 135/91) + Pilot (Personal)
**Workers:** AV-001 to AV-038 (Operator), AV-P01 to AV-P06 (Pilot)
**Session:** 25 (scaffold complete)

## Operator Suite
38 workers covering certification, maintenance, flight ops, safety, revenue, and compliance.
Commission model for enterprise operators. Subscription for individual workers.

## Pilot Suite
6 workers: Digital Logbook, Currency Tracker, My Aircraft, Training & Proficiency, Flight Planning, Alex Personal.
Free tier + Pro ($29/mo) + Pro+ ($49/mo).

## Special Programs
- Student pilots: Free while enrolled (STUDENT coupon)
- CFI/CFII academy instructors: Free Pro+ (CFIACADEMY coupon)
- Graduation grace: GRADPILOT → 3 months free

See `/docs/platform/pricing.md` for full pricing details.
