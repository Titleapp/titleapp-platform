# IR Workflow Catalog

## Investor Onboarding
**Trigger:** New investor expresses interest
**Steps:** Identity verification → Accreditation check → NDA/Disclaimer → Data room access → Deal presentation
**SOP:** accreditation_sop_v0

## Deal Creation
**Trigger:** GP creates new investment opportunity
**Steps:** Define deal type → Set terms → Configure waterfall → Upload docs → Compliance check → Activate

## Capital Raising
**Trigger:** Deal activated for fundraising
**Steps:** Investor outreach → Subscription processing → Accreditation verification → Countersign → Fund capital call

## Capital Call
**Trigger:** GP initiates capital call
**Steps:** Calculate pro-rata allocation → Generate notices → Send to investors → Track payments → Confirm funding

## Distribution
**Trigger:** Distributable event (sale, refi, cash flow)
**Steps:** Calculate net proceeds → Run waterfall → Allocate to investors → Generate notices → Process payments
**SOP:** distribution_sop_v0

## Quarterly Reporting
**Trigger:** End of quarter
**Steps:** Compile deal updates → Calculate performance (IRR, MOIC) → Generate report → Distribute to LPs

## Deal Exit
**Trigger:** Property sale, fund wind-down, or liquidity event
**Steps:** Calculate final proceeds → Run final waterfall → Final distribution → Generate K-1 data → Close deal
