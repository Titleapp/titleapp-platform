## Normal Checklist (Abbreviated)

### Pre-Flight Inspection

1. Documents -- check (AFM, W&B, airworthiness)
2. ELECTRIC MASTER -- OFF
3. ENGINE MASTER L and R -- OFF
4. Fuel quantity -- check visually and gauges
5. Walkaround -- complete (all panels, surfaces, tires, props, pitot, antennas)
6. Engine oil -- check (each engine, 4.5--5.0 L)
7. Coolant -- check (each engine)
8. Gear -- inspect (struts, tires, brakes)
9. Control surfaces -- free, correct sense, no damage
10. Baggage -- stowed and secured

### Engine Start

1. Safety harnesses -- fastened
2. Parking brake -- SET
3. ELECTRIC MASTER -- ON
4. Avionics -- ON (G1000 initialization)
5. ECU SWAP -- AUTOMATIC (both engines)
6. ENGINE MASTER L -- ON, wait for GLOW OFF
7. Power lever L -- IDLE
8. ELECTRIC MASTER -- START L (release when running)
9. Repeat for right engine
10. Engine instruments -- all green (both engines)
11. Generators -- check ON (both)

### Before Takeoff

1. Brakes -- hold
2. Engine run-up -- check both engines
3. ECU A/B check -- both engines
4. Propellers -- exercise (cycle)
5. Flaps -- T/O position
6. Trim -- set takeoff
7. Fuel -- sufficient, selectors correct
8. Instruments -- set (altimeter, DG, HSI)
9. Transponder -- set
10. Doors/canopy -- closed, locked (DOOR light OFF)
11. Safety harnesses -- secure

### Takeoff

1. Power levers -- MAX (both)
2. Engine instruments -- check
3. Brakes -- release
4. At Vr (71 KIAS) -- rotate
5. Positive rate -- gear UP
6. Above safe altitude -- flaps UP
7. Climb at Vy (85 KIAS) or cruise climb 95 KIAS

### Cruise

1. Power -- set as desired
2. Fuel management -- monitor quantities, balance L/R
3. Engine instruments -- all green
4. Lean mixture -- N/A (FADEC auto)

### Approach and Landing

1. ATIS/weather -- obtain
2. Fuel -- check
3. Below Vle (152 KIAS) -- gear DOWN, 3 green
4. Flaps -- APP, then LDG
5. Airspeed -- 80 KIAS on final
6. Power -- as required for glideslope
7. Touchdown -- main gear first, lower nose gently
8. Brakes -- as required

### Engine Shutdown

1. Power levers -- IDLE (cool-down 2 min)
2. Avionics -- OFF
3. ENGINE MASTER L -- OFF
4. ENGINE MASTER R -- OFF
5. ELECTRIC MASTER -- OFF

## Emergency Checklist

### Engine Failure During Takeoff (before Vr)

1. Power levers -- IDLE (both)
2. Brakes -- maximum
3. Emergency fuel valves -- OFF (both)
4. ENGINE MASTER -- OFF (both)

### Engine Failure After Takeoff (single engine)

1. Airspeed -- maintain Vyse (82 KIAS) minimum
2. Identify dead engine (dead foot = dead engine)
3. Operative engine -- MAX power
4. Dead engine -- confirm, feather propeller
5. Flaps -- UP
6. Gear -- UP (if safe altitude)
7. If unable to climb -- emergency landing straight ahead

### Engine Failure in Flight (single engine)

1. Airspeed -- maintain Vyse (82 KIAS)
2. Identify dead engine
3. Dead engine -- power lever IDLE
4. Dead engine -- propeller FEATHER
5. Dead engine -- fuel OFF, engine master OFF
6. Live engine -- power as required
7. Trim for single-engine flight
8. Declare emergency, divert to nearest suitable airfield

### Both Engines Failed

1. Airspeed -- 85 KIAS (best glide, props feathered)
2. Both propellers -- FEATHER
3. Select landing field
4. Attempt restart if altitude permits
5. Execute emergency landing per procedures

### Fire in Flight -- Engine

1. Affected engine -- fuel OFF, engine master OFF
2. Propeller -- FEATHER
3. Cabin heat -- OFF
4. Select landing field
5. Land as soon as possible

### Fire in Flight -- Electrical

1. AVIONIC MASTER -- OFF
2. ELECTRIC MASTER -- OFF
3. Emergency battery -- ON (if installed)
4. Cabin heat -- OFF
5. Ventilation -- open
6. Land at nearest airfield

### Spin Recovery

1. Power levers -- IDLE (both)
2. Rudder -- full AGAINST spin direction
3. Control stick -- fully FORWARD
4. Ailerons -- NEUTRAL
5. Flaps -- UP
6. When rotation stops -- neutralize rudder, recover from dive

## Systems Summary

### Engines (Austro Engine AE 300)
Each AE 300 is a 4-cylinder, turbocharged, liquid-cooled diesel engine with dual-channel FADEC. Burns JET A1 fuel. Single power lever per engine controls both power and propeller RPM. Reduction gearbox drives constant-speed, full-feathering propeller.

### Fuel System
Two wing tanks (L and R), each with MAIN and AUX sections. Gravity feed from MAIN to engine. Electric transfer pump moves fuel from AUX to MAIN. Emergency crossfeed valve available. Fuel gauges on G1000 MFD.

### Electrical System
28V DC system with two engine-driven generators (one per engine). Main battery + emergency battery. Essential bus powered by either generator or battery. G1000 avionics on dedicated avionics bus.

### Landing Gear
Retractable tricycle gear, hydraulically actuated. Emergency gear extension via nitrogen bottle. Gear position indicated by 3 green lights (down and locked). Gear warning horn sounds if gear not down with power below threshold.

### Flight Controls
Conventional stick and rudder. Electric pitch trim. Electric flaps (UP, APP, LDG). Rudder trim (manual). Roll trim not installed.
