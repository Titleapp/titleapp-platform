## 1. Introduction

This SOP provides aircraft-specific procedures for the Pilatus PC-12/47E (PC-12 NG) in single-pilot Part 135 operations. It supplements the General Fixed-Wing SOP and the PC-12 AFM. In any conflict, the AFM takes precedence.

### 1.1 CAS Response Procedures

Reference: General FW SOP Section 5 and PC-12 AFM Emergency/Abnormal Procedures.

### 1.2 Stabilized Approach Criteria

Reference: General FW SOP Section 7.1 and GOM.

**PC-12 Specific Stabilized Approach Speeds:**
- Target approach speed: DSB (defined speed bugs) per the AFM for the given weight and configuration
- Speed tolerance: DSB +/- 5 knots
- If not stabilized by the applicable gate, execute a go-around

### 1.3 Flap Speed Schedule

| Configuration | Max Speed (KIAS) | Buffer Notes |
|---------------|-----------------|--------------|
| Flaps 15 | Per AFM placard | 5 kt buffer recommended below Vfe |
| Flaps 30 | Per AFM placard | 5 kt buffer recommended below Vfe |
| Flaps 40 | Per AFM placard | 5 kt buffer recommended below Vfe |
| Gear Extension | 180 KIAS (Vlo) | Extend below 180 KIAS |

## 2. Preflight

### 2.1 Interior Preflight -- Expanded Checklist

**Cockpit Setup:**
- Circuit Breakers -- ALL IN (scan both panels)
- Battery 1 -- ON
- Battery 2 -- ON
- EPS -- CHECK/ON
- Standby Bus -- ON
- Avionics Master -- ON
- Condition Lever -- CUTOFF
- PCL -- IDLE
- Fuel Selectors -- verify
- Oxygen System -- CHECK (crew and passenger supply, pressure)
- Emergency Equipment -- CHECK (fire extinguisher, first aid, ELT)
- Logbook -- REVIEW (current entries, discrepancies, MELs)
- EFB -- POWERED, CURRENT
- Seat/Belts/Harness -- ADJUSTED, SECURE

**Cabin:**
- Emergency exits -- VERIFY operation and markings
- Seats/Belts -- CHECK secure
- Cabin equipment -- SECURE (all loose items stowed)
- Medical equipment -- VERIFY secure (air medical configuration)
- Cargo -- VERIFY secure (cargo configuration)

### 2.2 Exterior Preflight -- Walk-Around

Perform a systematic walk-around per AFM. Key inspection areas:

**Left Side:**
- Main landing gear -- tire condition, brake wear, strut inflation
- Left wing -- leading edge, de-ice boots (condition), fuel cap (secure), lights
- Left engine cowl -- oil level (check sight gauge), leaks, cowl fasteners
- Propeller -- blade condition, spinner, de-ice boots

**Empennage:**
- Horizontal stabilizer -- condition, de-ice boots
- Vertical stabilizer -- condition
- Trim tabs -- freedom of movement, condition
- Tail cone -- ELT antenna, static ports

**Right Side:**
- Right wing -- leading edge, de-ice boots, fuel cap, lights, pitot tube
- Main landing gear -- tire, brake, strut
- Inertial separator door -- verify position

**Nose:**
- Nose gear -- tire condition, steering linkage
- AOA vanes -- condition, covers removed
- Pitot tubes -- covers removed
- Static ports -- clear
- Windshield -- clean, no damage

**Main Cabin Door:**
- Door seals -- condition
- Locking mechanism -- proper operation
- Indicator windows -- verify green when latched

## 3. Normal Procedures -- Flows and Checklists

### 3.1 Before Start

**Flow:**
- Parking Brake -- SET
- Fuel Quantity -- CHECKED (sufficient for planned flight)
- Condition Lever -- CUTOFF
- PCL -- IDLE
- Ignition -- verify
- Seat Belts / Harness -- ON
- Avionics -- SET
- Transponder -- SET (ALT)
- EFB -- CONFIGURED for flight
- Oxygen -- AS REQUIRED
- Door -- CLOSED and LOCKED (verify indicator windows)

**Expanded Checklist:**
- Parking Brake -- SET
- Seats/Belts/Harness -- ADJUSTED/SECURE
- Trim -- SET FOR TAKEOFF (3 GREEN in electronic trim display)
- Circuit Breakers -- ALL IN
- Fuel Quantity -- CHECKED
- Ignition / Start -- AS REQUIRED
- Emergency Equipment -- CHECKED
- Pressurization -- SET
- Avionics/Radios -- SET
- Transponder -- ALT
- Door -- CLOSED/LOCKED

### 3.2 After Start

**Flow:**
- ITT / Ng -- STABILIZED within limits
- Oil Pressure -- CHECK (green range)
- Generators -- ON, CHECKED
- Avionics -- VERIFY powered
- CAS -- CLEAR (no warnings/cautions)

**Expanded Checklist:**
- Engine Parameters -- CHECKED (ITT, Ng, Oil Pressure, Oil Temp)
- Generators 1 & 2 -- ON
- CAS -- CLEAR
- Avionics -- CHECKED
- Trims -- CHECKED (3 GREEN)
- Flight Controls -- FREE and CORRECT
- Flaps -- CHECK operation
- Condition Lever -- GROUND IDLE
- Pressurization -- SET

### 3.3 Taxi

**Flow:**
- Clearance -- OBTAINED (IFR) or intentions communicated (VFR)
- Taxi Lights -- ON
- Brakes -- CHECK (initial taxi)
- Nosewheel Steering -- CHECK
- Flight Instruments -- CHECK (heading, attitude aligning)

**Callouts:**
- If clearance is in doubt: stop and clarify
- Wing clearance minimum: 10 ft from any object
- Announce taxi route if complex

### 3.4 Before Takeoff

**Flow:**
- Run-Up area -- STOPPED
- Flight Controls -- FULL DEFLECTION CHECK
- Trim -- SET FOR TAKEOFF
- Flaps -- SET (normally 15 or 30)
- Instruments -- CHECKED
- Departure Brief -- COMPLETE (departure procedure, engine failure plan, abort criteria)
- Transponder -- ALT
- Pitot Heat -- ON
- Lights -- SET (landing, recognition, strobe)
- Condition Lever -- MAX RPM
- PCL -- CHECK response

**Expanded Checklist:**
- Departure Brief -- COMPLETE
- Seats/Belts -- SECURE
- Flight Controls -- FREE and CORRECT
- Trims -- SET FOR TAKEOFF / 3 GREEN
- Flaps -- SET FOR TAKEOFF
- Instruments/Avionics -- SET
- Fuel Quantity -- RECHECKED
- Condition Lever -- MAX RPM
- Pitot/AOA/Stall Heat -- ON
- Lights -- SET
- Transponder -- ALT
- Cabin/Passengers -- SECURE
- Inertial Separator -- AS REQUIRED
- Door -- CONFIRMED LOCKED

### 3.6 Line-Up and Takeoff

**Flow:**
- Heading -- VERIFY aligned with runway
- HSI -- MATCHES runway heading
- Flight Director / FMS -- SET for departure
- Lights -- ALL ON (landing, pulse/recognition, strobe)
- PCL -- ADVANCE to takeoff power

**Callout:** "Torque [XX] PSI" (verify takeoff power set)

**Takeoff Roll:**
- "Airspeed alive"
- At Vr (approximately 76 KIAS at max gross): "Rotate"
- "Positive rate -- gear up"

### 3.7 After Takeoff / Climb

**Climb Speeds:**

| Altitude | Cruise Climb (KIAS) | Best Rate Vy (KIAS) |
|----------|-------------------|---------------------|
| Sea Level | 180 | 130 |
| 5,000 ft | -- | 125 |
| 10,000 ft | 160 | -- |
| 15,000 ft | -- | 125 |
| 20,000 ft | 140 | 120 |

**Power Settings:**
- Takeoff: 850 C ITT maximum for 5 minutes
- After 5 minutes: reduce to 820 C ITT (max continuous)
- Cruise climb: 780 C ITT target

**Flow (After Takeoff):**
1. Gear -- UP (positive rate of climb)
2. Inertial Separator -- CLOSED above 3,000 ft AGL (bypass in icing conditions below 10 C)
3. Yaw Damper -- ON
4. Lights -- AS REQUIRED (landing lights off above traffic pattern)
5. Flaps -- UP at 100 KIAS minimum
6. Autopilot -- ON above 400 ft AGL

**Expanded Checklist:**
- Gear/Flaps -- UP / 0
- Yaw Damper -- ON
- Inertial Separator -- AS REQUIRED
- Climb Power -- SET (820 C ITT max continuous)
- Cabin Altitude/Pressure/Rate -- CHECKED (LOW CAB mode preferred)
- Lights/Ice Protection -- AS REQUIRED
- Passengers/Crew -- CHECKED

**Callouts:**
- "No more usable runway" (during takeoff roll)
- "100 knots, flaps up"
- "400 feet, autopilot on"
- "3,000 AGL, inertial separator closed"

### 3.8 10,000 ft Climbing

**Flow:**
- Recognition Lights -- OFF (or as required above 10,000 ft)
- 10K Checklist -- VERIFY

**Expanded Checklist:**
- Recognition Lights -- OFF / AS REQUIRED below 10,000 ft
- Pressurization -- CHECKED (differential increasing as expected)

**Callouts:**
- "10,000, recognition lights off"
- "Pressurization differential increasing"
- "How is the temperature in the cabin?" (crew coordination)

### 3.9 Cruise

**Power Setting:**
- Maximum cruise: per QRH (37 PSI torque / 780 C ITT maximum)
- Reference QRH for altitude-specific power tables

**Cruise Altitude Rule of Thumb:**
- Total leg distance (NM) / 6 = approximate cruise altitude in thousands of feet

**Expanded Checklist:**
- Power -- SET (per QRH, torque less than 37 PSI, ITT less than 780 C)
- Engine Gauges -- CHECKED (all in green)
- Passengers/Crew -- CHECKED
- Cabin Altitude -- ADVISE crew of current cabin altitude

### 3.10 Descent

**VNAV Setup (FMS Equipped):**
1. Verify fuel remaining
2. Calculate/verify TOD (Top of Descent)
3. Check VSD (Vertical Situation Display)
4. Confirm cruise altitude
5. Confirm waypoint altitude constraints

**VNAV Capture:**
- VTA (Vertical Track Alert): "1 minute to TOD"
- Verify VDI (Vertical Deviation Indicator) is Cyan or White
- VPTH mode: aircraft follows computed vertical path
- SPD mode: aircraft maintains target speed

> **CAUTION**: In VPTH mode, there is NO overspeed protection. Monitor airspeed carefully during descent.

**Night IFR Cancellation Criteria:**
- Within 15 NM of the airport
- Airport in sight
- Ceiling at or above 5,000 ft

**Expanded Checklist:**
- VNAV -- VERIFIED (TOD, constraints, VSD)
- Ice Protection -- AS REQUIRED
- Altimeters -- SET (local altimeter below FL180)
- Landing Performance -- VERIFIED (EFB tools, runway condition assessment, FICON levels if applicable)

**Callouts:**
- "VTA -- 1 minute to top of descent"
- "Pre-approach: VDI is cyan [or white]"

### 3.11 Approach

### 3.11.1 General

- Maintain 150 KIAS or less within 10 NM of the airport
- Stabilized approach criteria per GOM and General FW SOP

**RNAV Circling Approaches:**
- The FMS defaults the missed approach point to Category D
- The pilot MUST update to Category B speeds and minimums

> **NOTE**: APEX limitations -- LOC and LDA approaches may not be executed with FMS overlay. Verify approach type compatibility.

### 3.11.2 Precision Approach Sequence

1. Terminal area / vectors: HDG selected, NAV preview, Flaps 0, 150 KIAS, PCL 15 PSI
2. Intercept glideslope at or outside the FAF
3. 3 miles from FAF: APR Armed, LOC Armed/Captured, GS Armed, Flaps 15, Gear Down one dot above glideslope
4. At FAF: Landing flaps set, altitude set for missed approach, airspeed DSB
5. At DH: Runway not in sight = Missed Approach; Landing assured = disengage AP/YD at 200 ft AGL minimum

### 3.11.3 Non-Precision Approach Sequence

1. Terminal: NAV Armed/Captured, Pitch Mode, Flaps 0, 150 KIAS, PCL 15 PSI
2. 1.5 NM from FAF: Flaps 15
3. At/0.5 NM from FAF: Gear Down, pitch descend to MDA/DA, landing flaps set, airspeed DSB
4. At MDA: Altitude set for missed approach; Runway not in sight = MAP; Landing assured = disengage AP/YD at 400 ft AGL minimum
5. Circling: Maintain 105 KIAS with Flaps 15 until base leg

### 3.11.4 VFR Approach

Normal traffic pattern: 1,500 ft AGL, 150 KIAS, Flaps 0. Refer to graphical maneuver profiles below.

**Expanded Checklist:**
- FMS Approach -- LOADED
- Approach Brief -- COMPLETE
- Circling: verify Category B speeds if applicable
- Missed Approach -- BRIEFED
- Minimums -- BARO SET
- TAWS -- CONSIDERED
- Lights/Ice -- SET
- Passengers/Cabin -- CHECKED/SECURED
- Inertial Separator -- OPEN (below 10,000 ft for approach)

**Callouts:**
- "Approach loaded"
- "Bugged and briefing complete"
- "Checklist complete"
- "APR armed"
- "Final approach course captured"

### 3.12 Before Landing / Landing

**Default Configuration:** Flaps 40 (Flaps 30 acceptable per pilot preference or conditions)

**Short Field Check at 500 AGL:**
- Gear -- DOWN (3 GREEN)
- On DSB airspeed
- Flaps -- 40
- On glideslope/VGSI

**Flow:**
1. PCL -- BETA/REVERSE after touchdown
2. Condition Lever -- GROUND IDLE
3. Brakes -- apply after weight on wheels, at approximately 50 KIAS

> **CAUTION**: Do NOT back up using reverse thrust. Reverse is for deceleration only.

**Expanded Checklist:**
- Gear -- DOWN / 3 GREEN (with 5 kt speed buffer above Vle)
- Flaps -- SET FOR LANDING (30 or 40)
- Pressurization -- less than 0.7 PSI differential
- AP/YD -- OFF / OFF

> **CAUTION**: On circling approaches, verbalize the missed approach procedure before descending below MDA.

**Callouts at 500 AGL:**
- "Gear down, 3 green"
- "Flaps, final setting"
- "Yaw damper plan [on/off]"
- "Rudder trim centered"

### 3.13 After Landing

**Flow:**
1. Ice Protection -- SET (as required for taxi)
2. Lights -- SET FOR TAXI
3. Flaps -- UP (0)
4. Trim -- CENTERED / SET FOR TAKEOFF

**Expanded Checklist:**
- Radar -- STANDBY
- Flaps -- 0
- Condition Lever -- GROUND IDLE
- Trims -- 3 GREEN
- Ice Protection -- SET / OFF
- Lights -- SET FOR TAXI
- Flight Plan -- CLOSED (if applicable)

### 3.14 Taxi (Post-Landing)

**Callouts:**
- If clearance is in doubt, ask the crew or stop and verify
- Wing clearance minimum: 10 ft
- Request marshalling if needed at the ramp

Coordinate with the OCC for base arrivals.

### 3.15 Shutdown

**Flow:**
1. Inertial Separator -- OPEN (for ground operations)
2. Allow ITT to stabilize for 2 minutes at ground idle
3. Condition Lever -- CUTOFF
4. Electrical -- manage per shutdown sequence

**Battery Shutdown Order (SEBB):**
1. Standby Bus -- OFF
2. EPS -- OFF
3. Battery 2 -- OFF
4. Battery 1 -- OFF

**PCL Cover:**
- YELLOW cover = Aircraft is AIRWORTHY
- RED cover = Aircraft is NOT AIRWORTHY (RED cover removed only by maintenance)

**Post-Flight:**
- Complete 360-degree walk-around
- Check oil level (sight gauge) and tires
- Record any maintenance issues in the logbook
- Note any CAS messages that appeared during the flight

> **CAUTION**: Aircraft/Engine Exceedance CAS messages require maintenance follow-up before next flight. Do not dispatch with an unresolved exceedance.

- Clean the aircraft cabin (medical biohazard cleaning is NOT pilot responsibility)
- If stopping for more than 1 hour: install prop ties, pitot covers, AOA covers

**If overnight away from base:**
- Chock, gust lock, secure all doors
- Pin emergency exit
- Hangar if weather threatens, or install tail stand if left outside

**Expanded Checklist:**
- Parking Brake -- SET
- Lights/Ice -- OFF
- ITT -- STABILIZED 2 MIN at ground idle
- Condition Lever -- CUTOFF
- Fuel Quantity / HOBBS -- NOTED
- CAS -- NOTED
- Batteries -- AS REQUIRED (SEBB shutdown order)
- PCL Cover -- INSTALLED (Yellow = airworthy, Red = not airworthy)

## 4. Graphical Aircraft Maneuver Profiles

### 4.1 Takeoff -- Flaps 30 (Normal)

**Phase 1 -- Line Up:**
- Align with runway centerline
- Verify heading and HSI match

**Phase 2 -- Rotate:**
- Vr approximately 76 KIAS (at 10,450 lbs gross weight)
- Positive rate of climb: Gear UP

**Phase 3 -- Initial Climb:**
- Vx (obstacle clearance) or Vy (best rate)
- Flaps 15 for obstacle clearance if required

**Phase 4 -- Climb Check:**
- ITT 820 C max continuous
- Complete after-takeoff flow

### 4.2 VFR Traffic Pattern -- Normal Landing

**Downwind (1,500 ft AGL):**
- 150 KIAS, Flaps 0
- Disengage AP

**Abeam Threshold:**
- Gear DOWN
- Flaps 15
- PCL 15 PSI
- 110-120 KIAS

**Base:**
- Flaps as required
- YD disengaged
- 110-120 KIAS

**Final:**
- Flaps as required (30 or 40)
- Airspeed DSB

**After Landing:**
- PCL Idle, Reverse
- Braking, Condition Lever Ground Idle

> **Notes:**
> - Flaps 15 with Pusher Ice Mode = 110 KIAS minimum
> - Flaps 0 with boot failure = 137 KIAS minimum
> - Disengage AP below 1,000 ft AGL

### 4.3 VFR Traffic Pattern -- Short Field Landing

Same as normal pattern but:
- Flaps 40 on final
- DSB airspeed (target speed for weight)
- 500 AGL check: Gear Down 3 Green, On DSB, Flaps 40, On Glideslope

### 4.4 Precision Approach (ILS/GLS)

1. Terminal / Vectors: HDG selected, NAV preview, Flaps 0, 150 KIAS, PCL 15 PSI
2. Base / 3 NM from FAF: APR Armed, LOC Armed/Captured, GS Armed, Flaps 15, Gear Down (one dot above GS)
3. At FAF: Landing flaps set, Altitude set for Missed, Airspeed DSB
4. At DH: Runway not in sight = MAP; Landing assured = disengage AP/YD 200 ft AGL minimum

### 4.5 Non-Precision Approach

1. Terminal: NAV Armed/Captured, Pitch Mode, Flaps 0, 150 KIAS, PCL 15 PSI
2. Procedure Turn Inbound: Altitude set, Pitch descend, Flaps 15 at 1.5 NM from FAF
3. At FAF: Gear Down 0.5 NM, Pitch descend to MDA/DA, Landing flaps, Airspeed DSB
4. At MDA: Altitude set for Missed; Runway not in sight = MAP; Landing assured = disengage AP/YD 400 ft AGL minimum
5. Circling: maintain 105 KIAS with Flaps 15 until base leg

> **NOTE**: APEX limitations -- LOC and LDA approaches may not be executed with FMS overlay.

### 4.6 Go-Around / Missed Approach

1. Press G/A switch
2. PCL set to balked landing torque
3. Climb airspeed 85 KIAS
4. Flaps CHECK / SET 15
5. Positive rate: climb airspeed 95 KIAS (Pusher Normal mode)
6. Gear UP
7. Yaw Damper ON
8. Above 100 KIAS: Flaps 0
9. Ice protection as required

### 4.7 Maximum Rate Descent

1. PCL -- IDLE
2. Landing Gear -- DOWN (below 180 KIAS)
3. Aircraft Speed -- Vmo (240 KIAS / 0.48 M)
4. All Occupants -- check seat lap and shoulder belts
5. Oxygen Mask -- don before cabin altitude exceeds 10,000 ft
6. Passenger Oxygen -- AUTO (Switch Auto, Main Oxygen Confirm ON, verify MFD "PAX OXY" advisory, passengers don masks)
7. Left Windshield Heat -- AS REQUIRED

> **CAUTION**: In turbulence, reduce speed to 170 KIAS.

**Recovery:**
1. Landing Gear -- UP (below 180 KIAS)
2. PCL -- Increase as required
3. Altimeter -- SET below transition level
4. NAV/HDG -- As required

### 4.8 Approach to Stall and Recovery -- Landing Configuration

> **WARNING**: Stall must be avoided when the stick pusher is inoperative. Excessive wing drop and altitude loss may result during stall with flaps down and/or when power is applied. Stalls should be avoided at all times.

> **CAUTION**: Stall speeds in turns are higher. AOA may not be reliable. If CAS Pusher illuminates, terminate the maneuver.

**Entry (5,000 ft AGL minimum):**
1. PCL -- SET 15 PSI
2. AP/YD/FD -- OFF
3. Landing Gear -- DOWN (below 180 KIAS)
4. Flaps -- 40 (below 130 KIAS)
5. Allow aircraft to slow while maintaining altitude

**At Stick Shaker Activation:**
1. PCL -- SET takeoff torque
2. Pitch -- Adjust nose to zero pitch reference line
3. As DSB centers -- Initiate climb
4. Flaps -- 15
5. Landing Gear -- UP (positive ROC)
6. Flaps -- 0 (above 100 KIAS)

> Note: Use rudder to aid in leveling wings and reduce adverse yaw from aileron deflection.

**Completion:**
- Altitude: Level off at starting altitude +/- 100 ft
- Airspeed: 150 KIAS +/- 10 KIAS
- Maneuver Complete

### 4.9 Approach to Stall and Recovery -- Clean Configuration

**Entry (5,000 ft AGL minimum):**
1. PCL -- SET 15 PSI
2. AP/YD/FD -- OFF
3. Landing Gear -- UP
4. Flaps -- 0
5. Allow aircraft to slow while maintaining altitude

**At Stick Shaker:**
1. PCL -- SET takeoff torque
2. Pitch -- Zero pitch reference
3. As DSB centers -- climb
4. Verify -- Flaps 0 and Landing Gear UP

**Completion:**
- Level off at starting altitude +/- 100 ft
- 150 KIAS +/- 10 KIAS

### 4.10 Approach to Stall and Recovery -- Departure Configuration

**Entry (5,000 ft AGL minimum):**
1. PCL -- SET 15 PSI
2. AP/YD/FD -- OFF
3. Landing Gear -- UP
4. Flaps -- 15 (below 165 KIAS)
5. PCL -- IDLE
6. Commence Turn -- 15 degree bank

**At Stick Shaker:**
1. PCL -- SET takeoff torque
2. Bank -- Roll wings level
3. Pitch -- Zero pitch reference
4. As DSB centers -- climb
5. Flaps -- 0 (above 100 KIAS)

### 4.11 Steep Turns

**Entry (5,000 ft AGL minimum):**
1. Cruise check -- complete
2. Torque -- as required to maintain airspeed and altitude
3. Airspeed -- 150 KIAS +/- 10
4. Bank angle -- roll smoothly to 45 degrees
5. Altitude -- maintain constant +/- 100 ft

**Exit:**
1. Airspeed -- 150 KIAS +/- 10
2. Bank -- roll smoothly to wings level
3. Altitude -- maintain constant +/- 100 ft
4. Cruise check -- complete

**Option (after 180 degree heading change):**
Roll smoothly to 45 degrees in the opposite direction, maintaining same parameters.

### 4.12 Unusual Attitudes

**Nose Low (above 5,000 ft AGL):**
1. Attitude -- verify nose low using reliable cues
2. PCL -- reduce power initially to avoid excessive altitude loss
3. Bank Angle -- roll smoothly to wings level
4. Pitch -- smoothly raise nose to at or above horizon
5. PCL -- add power as required to climb to safe altitude

**Nose High (above 5,000 ft AGL):**
1. Attitude -- verify nose high using reliable cues
2. PCL -- add power as required to avoid excessively slow airspeed
3. Bank Angle -- can be used to aid in lowering nose to the horizon, then smoothly roll to wings level
4. Pitch -- smoothly lower nose to maintain airspeed above stall
5. PCL -- as required to attain and maintain 150 KIAS +/- 10

## 5. Overwater Operations

> **Mission Type: Air Medical / Charter / Cargo** (applicable to all overwater flights)

### 5.1 Required Equipment

For flights beyond autorotation/glide distance from shore:
- Life vests for all occupants
- HEEDS bottles (Helicopter Emergency Egress Device System) or equivalent
- Personal Locator Beacons (PLBs) as required
- Extended Life Over-water Transmitter (ELOT): 30-minute minimum

### 5.2 Procedures

- Brief all occupants on life vest location and operation
- Brief ditching procedures
- Monitor weather and sea state
- Maintain altitude sufficient for maximum glide range to nearest land
- Report position to the OCC at required intervals
