# Florida Real Estate Licensing
# Path: raas/real-estate/FL/licensing.md

---

## License Types

### Sales Associate
- 63-hour pre-license course
- State exam
- Must work under a licensed broker
- Renewal: every 2 years
- CE: 14 hours per renewal cycle (includes 3 hours ethics/law, 3 hours core, 8 hours specialty)

### Broker
- 24 months as active sales associate
- 72-hour broker pre-license course
- State exam
- Can operate independently, employ sales associates
- Same CE requirements

### Broker Associate
- Holds broker license but works under another broker
- Same qualifications as broker

## Display Requirements
- AI must include on all advertising: agent name, license number (optional but recommended), brokerage name
- Brokerage name must be the registered name with DBPR

## Continuing Education
- AI tracks CE deadlines per agent/team member
- Alert 90 days before expiration
- Alert 30 days: "Your license expires [date]. You need [X] CE hours."

## Property Management Licensing
- Florida requires a real estate license to manage property for others for compensation
- Exception: salaried employees of a property owner
- Community Association Managers (CAM) have a separate license for HOA/condo management
