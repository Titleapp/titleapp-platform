# RAAS — Homeowners Associations (CA)

## Purpose
This directory defines the **Rules as a Service (RAAS)** for Homeowners Association (HOA)
operations supported by TitleApp in California.

---

## HOA Scope Covered
- HOA disclosures during sale or lease
- Governing document handling (CC&Rs, bylaws)
- Transfer and resale packages
- HOA fee obligations and status checks
- Management company interactions

---

## Core RAAS Responsibilities
- Ensure HOA disclosure requirements are met
- Flag HOA-encumbered properties
- Prevent unsupported HOA actions
- Guide compliant transaction flows involving HOAs

---

## Regulatory Sources (Reference Only)
- California Civil Code (Davis-Stirling Act)
- California Department of Real Estate
- HOA governing documents (client-provided)

> ⚠️ HOA rules vary by association.  
> Client-specific documents override defaults.

---

## Update Policy
- Reviewed quarterly
- Immediate updates for statutory HOA changes
- Client overrides allowed but logged
